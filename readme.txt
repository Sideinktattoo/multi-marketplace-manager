WooCommerce için çoklu tedarikçi ve pazar yeri entegrasyonu + kar/zarar hesaplama eklentisi.

== Description ==

Multi Marketplace Manager eklentisi WooCommerce sitenizi çoklu tedarikçiler ve pazar yerleri ile entegre etmenizi sağlar.

Özellikler:
- Çoklu tedarikçilerden XML üzerinden ürün importu
- Trendyol, Hepsiburada, n11 gibi pazar yerleriyle entegrasyon
- Otomatik fiyat hesaplama (maliyet + kar marjı)
- Çoklu pazar yerine otomatik ürün gönderimi
- Pazar yerlerinden otomatik sipariş çekme
- Kargo takip sistemi
- Detaylı kar/zarar raporları

== Installation ==

1. Eklenti ZIP dosyasını indirin
2. WordPress admin panelinizde Eklentiler > Yeni Ekle > Yükle'ye gidin
3. ZIP dosyasını yükleyin ve etkinleştirin
4. WooCommerce > Multi Marketplace Manager bölümünden ayarları yapılandırın

== Frequently Asked Questions ==

= Tedarikçi XML formatı nasıl olmalıdır? =
Tedarikçi XML'i standart bir ürün listesi formatında olmalıdır. Örnek format için dokümantasyona bakın.

= Hangi pazar yerleri destekleniyor? =
Şu an Trendyol, Hepsiburada ve n11 desteklenmektedir. Diğer pazar yerleri için talepte bulunabilirsiniz.

= Ürünler ne sıklıkla senkronize edilir? =
Varsayılan olarak her 6 saatte bir senkronizasyon yapılır. Bu ayar değiştirilebilir.
